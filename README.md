Socialize
========

![socialize](http://media.giphy.com/media/bVhovKpr7wWs/giphy.gif)

An application that brings together all your social accounts

## Requirements

* NodeJS (#obvz)
* Bower

## Set Up

1. Set Up Environment Variables

  - SESSION_SECRET

  - BASE_URL
  
  For Facebook API:
  - FACEBOOK_APP_ID
  - FACEBOOK_APP_SECRET

  For Twitter API:
  - TWITTER_CONSUMER_KEY
  - TWITTER_CONSUMER_SECRET
  
  For Google API:
  - GOOGLE_REALM
  - GOOGLE_CLIENT_ID
  - GOOGLE_CLIENT_SECRET

  For Foursquare API:
  - FOURSQUARE_CLIENT_ID
  - FOURSQUARE_CLIENT_SECRET

  For NodeMailer:
  - MAILER_SERVICE
  - MAILER_EMAIL
  - MAILER_USERNAME
  - MAILER_PASSWORD
  
2. Run `npm install`

3. Run `bower install`

4. Edit `.hedgehog.js` to your heart's desire

5. Run `gulp`

6. Code!
